Calculate volume

Create a function called getVolume, that takes 3 Int arguments which supposed to be a length, a width and a height of some object and returns the volume of the parallelogram with these sides. If the third, or both the third and the second dimensions are not specified, they should be 1 by default.



Memory limit: 256 MB
Time limit: 8 seconds

Caution

You may see errors in your code or in execution results because of missing context. Don't mind those; simply write your solution and click Check.
