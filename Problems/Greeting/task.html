Greeting

Suppose you have a function that takes a name and a few other arguments and returns a greeting.

fun greetUser(
    name: String,
    admin: Boolean = false,
    smith: Boolean = false,
    honorific: String = "",
    greet: String = "Greetings"
): String {
    return if (!admin && !smith) {
        "$greet, $honorific $name"
    } else {
        "Matrix Error"
    }
}

Implement a greetNeo() function that returns the string "Hello, Mr. Anderson". Use the greetUser() function with the correct arguments.
